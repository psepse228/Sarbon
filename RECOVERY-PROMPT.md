## Возможная причина потери сессии (для расследования)

Ты пожаловался, что сессия Claude Code теряется при каждой перезагрузке компьютера. Это НЕ решено в этой сессии — нужно продиагностировать отдельно. Вопросы, которые стоит выяснить в следующий раз:
- Как именно ты открываешь Claude Code после перезагрузки — то же окно VS Code / та же папка проекта, или заново?
- Включена ли в VS Code настройка "Restore Windows" (восстановление последней сессии при запуске)?
- Используешь ли Windows Fast Startup (может мешать корректному сохранению состояния)?
- Если используешь CLI напрямую — используешь ли `claude --continue` или `--resume`, или каждый раз новую сессию?

Это уже отмечено как открытый вопрос в `hot.md` вики — реши его в следующей сессии, когда будет время.
